Digital Random Colouring Canvas

Hello! Welcome to my website where you can download my colouring game!

What is it?

In one of my classes I was tasked to create my very own Creative Coding Project using Processing and for that I decided to develop a colouring game for your computers or laptops.
Currently there are a total of 8 sketches which you can paint with 18 different colours and 2 different types of paint brushes. But if your want to create your own art, there is also a blank canvas option.

Unfornately, you will not be able to select the sketches you get as it is random.

Controls

Now here are some simple controls that you need to remember:

0. Starting the Game
To start the colouring game, after dowloading both Processing and the Game which can be found at the download section, open the processing file called "Colouring-In Game" and then press CRTL + R (Windows) or COMMAND + R (Mac).
1. Picking Paintbrush Type
There are two types of paintbrush you can use as of now. To select a paintbrush you must press downt:
  • Left Mouse Button for Circle
  • Right Mouse Button for Square
2. Colours
You can select a colour by clicking on them at the right hand side of the screen as seen below.
3. Painting
To paint all you need to do is hold down the mouse button (LEFT or RIGHT) and drag it along the screen.
4. Changing Size of Paint
This is done by using the slider on the further right of screen. The default is 25 but it can go as low as 0 and as high as 150.
5. Want to use a Blank Canvas?
To open up the blank canvas, you press the ENTER key button.
6. Getting a Random SKetch
On the blank cavans, to get a sketch to colour in you must click on the mouse and press ALT key at the same time.
7. Save Coloured Sketch
If you want to save what you've coloured you press the 's' key button. The image should then be in your Processing Sketch directory when you first open up the program.
Note: When you change sketch, the painting progress will reset and will no longer be like you left it. Sorry.
8. Instructuion Reminder
If you are lost, you can press the 'Help Icon' to get a quick reminder of some key controls.
However, if you want a more detailed looked at the controls, you can press the BACKSPACE key button.

If you don't remember all this, don't worry! When you first open the program, you will be welcomed with an instruction screen before you can start painting.
